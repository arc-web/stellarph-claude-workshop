# GitHub Basics

GitHub is where your team's work lives. You don't need to be a developer to use it. This is the minimum you need for today.

---

## What GitHub Is

Think of it as Google Drive for code - but with version history, collaboration tools, and the ability to share your work publicly or privately.

Today you'll use it to:
- Keep your pod's work in one place
- Share links instead of sending files
- Show what you built at the end

---

## Step 1 - Create an Account

Go to https://github.com and sign up. Free account is all you need.

---

## Step 2 - Create a Repository

A repository (repo) is a folder for your project.

1. Click the **+** in the top right corner
2. Click **New repository**
3. Name it something clear: `my-pod-project` or `workshop-build`
4. Set it to **Public** (so your team and facilitators can see it)
5. Check **Add a README file**
6. Click **Create repository**

---

## Step 3 - Edit Your README

Your README is the front page of your repo. GitHub renders it automatically.

1. Click the pencil icon on `README.md`
2. Write what your project is, what it does, and how to use it
3. Click **Commit changes** when done

That's a save.

---

## Sharing With Your Pod

Once your repo exists:
- Copy the URL from your browser
- Share it in your Discord pod channel
- Anyone with the link can view it (if it's public)

To let teammates edit it: Settings > Collaborators > Add people by their GitHub username.

---

## Basic Markdown (for your README)

GitHub READMEs use Markdown. Here's all you need:

```markdown
# Big heading
## Smaller heading

Normal paragraph text.

- Bullet point
- Another bullet

**bold text**
*italic text*

[Link text](https://url.com)

```code block```
```

## If You Build Something With Claude Code

Claude Code can push your code to GitHub automatically. Just tell it:

```
Push this to my GitHub repo at [your repo URL]
```

Or it will guide you through the steps.

---

## You Don't Need to Know Git

For today, the GitHub web editor is enough. If you want to go deeper later:
- **GitHub Desktop** - a visual app, no command line needed - https://desktop.github.com
- **Git basics** - https://docs.github.com/en/get-started/quickstart
