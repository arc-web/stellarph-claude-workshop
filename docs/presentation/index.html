How it works

📥
Write your prompt
You compose a task description, code snippet, or question in the workshop environment.
⚙️
Claude processes with visibility
Claude executes your request while the workshop shows you the exact tokens sent, the model's reasoning steps, and token usage breakdown.

You receive the result plus a transcript of what Claude considered, which parts of your context it used, and why it chose this approach.
